详细描述

本模块基于 open62541 开发

  • 提供了 OPC UA 服务器、客户端的封装，以及 OPC UA 支持的数据类型包括 Object、Variable、Method、View、Event 的定义。
  • 关于如何使用此模块，可参考 工业自动化通信协议 OPC UA 说明文档

传输协议

枚举值
UDP_UADP 

使用 UDP 传输协议映射和 UADP 消息映射的组合，此协议用于 无代理 的消息传递

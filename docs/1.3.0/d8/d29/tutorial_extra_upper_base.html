写在前面

首先要清楚顶层模块主要负责实际项目中各个功能逻辑的安排与操控，具有确定的可执行程序，一般就是 main.cpp 文件所产生。在 前世今生 了解到 SRVL 2.x 系列及之前版本，顶层模块主要由视觉库本身维护，但视觉库本身不应该管理庞大的逻辑需求，应该只提供各类功能即可。因此，用户在使用 RMVL 的时候需要自行实现顶层模块。

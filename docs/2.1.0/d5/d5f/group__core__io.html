非阻塞模式，即读取数据时不会等待，如果没有数据到来则立即返回 -1

◆ readCorners()

void rm::readCorners ( std::istream & in,
std::vector< std::vector< std::array< float, 2 > > > & corners )

#include <rmvl/core/io.hpp>

导入角点数据

  • 读取的文件形如以下内容
    1.9, 2.11,
    3.12, 4.13, 5.14, 6.15,
    ---
    7.16, 8.17,
    ---
  • 第一次调用 readCorners 时读取到的内容为 {(1.9, 2.11)} 和 {(3.12, 4.13), (5.14, 6.15)}
参数
[in]in输入流对象
[out]corners读取出的角点数据
示例

详细描述

IP 协议族，包含 IPv4 和 IPv6 以及多播选项的相关定义

  • 在网络层，提供 ip::Networkv4 和 ip::Networkv6 类来表示 IPv4 和 IPv6 地址
  • 在传输层，提供 ip::tcp 和 ip::udp 命名空间，分别表示 TCP 和 UDP 协议
  • 提供多播选项的封装类，便于在 Socket 上设置多播相关选项

详细描述

Web 应用程序框架

  1. 轻量级的异步 Web 应用程序框架，支持路由、中间件，支持路径参数和查询参数
  2. 所有设计均参考 Express.js，具体的 Express.js 教程请参考 https://expressjs.com/
  3. 目前支持的方法与 rm::Router 一致

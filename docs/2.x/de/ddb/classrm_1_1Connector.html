Socket 连接器

用于连接到远程或本地服务器，建立会话，常用于客户端

// 连接远程服务器
rm::Connector c1(rm::Endpoint(rm::ip::tcp::v4(), 8080), "127.0.0.1");
// 连接本地服务器
rm::Connector c2(rm::Endpoint(rm::ipc::packet(), "/tmp/socket"));

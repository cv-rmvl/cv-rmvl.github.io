详细描述

使用 cv::Matx 改写的轻量级卡尔曼滤波和扩展卡尔曼滤波模块

  • 考虑到 OpenCV 中提供的 cv::KalmanFilter 是基于 cv::Mat 实现的，cv::Mat 的内存操作在运行时是在堆上打开的，因此会消耗大量的时间，本模块使用 cv::Matx 来实现 KF 和 EKF 的功能，以达到轻量化的目的。
  • 相关知识点可参考说明文档 卡尔曼滤波 以及 扩展卡尔曼滤波

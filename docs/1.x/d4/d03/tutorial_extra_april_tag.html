AprilTag 数据集

AprilTag 视觉标签数据集采用机甲大师 RoboMaster S1 视觉标签的内容，包含 0~9 共 10 个阿拉伯数字，A~Z 共 26 个拉丁字母，这些均可在 rm::TagType 枚举类型中找到。

数据集的所有视觉标签均为 Tag25h9 格式，可在 Github 中搜索 apriltag-generation 寻找视觉标签的生成程序。 [1]

警告
  • 目前暂不支持任意修改标签类型，使用上请确保标签类型正确
  • 暂不支持除 Tag25h9以外的标签格式

1. 安装与配置

1.1 如何在 Windows 上使用 RMVL？

RMVL 的 2.1.1 版本起可直接在 Github 下载预编译的可执行文件，例如 2.1.1 版本可直接点击此处进行下载，下载完成后双击即可执行安装程序。

具体教程可参考此处。

1.2 如果一定要源码编译，怎么在 Windows 上启用 OPC UA 模块？

有两种方法，第一种需要安装 Git，在 CMake 配置时输入

cmake -DBUILD_OPEN62541=ON ..

RMVL 会自动从 Github 下载对应版本的 open6251，自动开启 WITH_OPEN62541，并将 opcua 模块添加至构建系统中。执行结束，若无报错则表示配置成功。

第二种方法需要自行下载 open62541 的源码，并进行编译，具体过程省略，这种方法需要在 CMake 配置时输入

cmake -Dopen62541_DIR=<path/to/open62541> ..

其中 <path/to/open62541> 为 open62541Config.cmake 文件的路径。

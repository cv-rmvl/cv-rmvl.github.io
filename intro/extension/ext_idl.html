【02】IDL 接口定义语言

约 67 字小于 1 分钟

写在前面

适用于 RMVL 的 Visual Studio Code 扩展提供了 *.para 文件的语法高亮、代码提示、悬浮提示与代码块。在安装此插件后，打开任一 *.para 文件即可激活此插件。

Linux 配置教程

324字约1分钟

Linux 平台目前推荐使用源码编译的方式安装 RMVL，如果是 Debian 系的发行版，比如 Ubuntu、Debian 也可以直接下载 Debian 包进行安装。

源码编译

前置环境安装

Debian 系发行版可使用以下命令行来获取 CMake

sudo apt install cmake cmake-qt-gui cmake-curses-gui

其他发行版类似，其中 cmake-qt-gui 和 cmake-curses-gui 分别是 Qt 版的 CMake GUI 程序和 命令行 版的 CMake GUI 程序。

安装 RMVL

在您觉得合适的地方打开终端，使用命令行输入

git clone https://github.com/cv-rmvl/rmvl.git
git checkout 2.1.1

提示

也可以直接从 GitHub 上下载 RMVL 源代码，选择你想要的版本对应的 Source code，下载解压后即可使用。

在 RMVL 项目根目录打开终端，构建并安装 RMVL

mkdir build
cd build
cmake ..
cmake --build . --parallel 8
cmake --install .

测试安装结果

打开终端，输入

rmvl_version

若显示对应版本号，则说明安装成功

直接下载 Debian 包

在 GitHub Release 页面下载对应版本的 Debian 包，使用以下命令安装

sudo dpkg -i RMVL-2.1.1-x86_64-linux.deb

安装完成后，打开终端，输入

rmvl_version

若显示对应版本号，则说明安装成功

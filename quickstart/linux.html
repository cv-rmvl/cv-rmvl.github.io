Linux 配置教程

约 391 字大约 1 分钟

一键安装

Linux 平台目前推荐使用 rdt 工具完成 RMVL 的安装，这里给出 rmvl/rdt 的一键安装脚本

wget https://cv-rmvl.github.io/install -qO - | bash

源码编译

如果您想要更深入地了解 RMVL 的安装过程，或是需要对 RMVL 进行二次开发，又或是想自由定制启用的 RMVL 功能，那么可以选择从源码编译安装 RMVL。

前置环境安装

Debian 系发行版可使用以下命令行来获取 CMake

sudo apt install cmake cmake-qt-gui cmake-curses-gui

其他发行版类似，其中 cmake-qt-gui 和 cmake-curses-gui 分别是 Qt 版的 CMake GUI 程序和 命令行 版的 CMake GUI 程序。

安装 RMVL

在您觉得合适的地方打开终端，使用命令行输入

git clone https://github.com/cv-rmvl/rmvl.git
git checkout 2.4.0

提示

也可以直接从 GitHub 上下载 RMVL 源代码，选择你想要的版本对应的 Source code，下载解压后即可使用。

在 RMVL 项目根目录打开终端，构建并安装 RMVL

mkdir build
cd build
cmake ..
cmake --build . --parallel 8
cmake --install .

测试安装结果

打开终端，输入

rmvl_version

若显示对应版本号，则说明安装成功

直接下载 Debian 包

在 GitHub Release 页面下载对应版本的 Debian 包，例如，在 Ubuntu 22.04 下，对于 2.4.0 版本，可以使用以下命令安装

sudo dpkg -i RMVL-2.4.0-x86_64-linux-u22.deb

安装完成后，打开终端，输入

rmvl_version

若显示对应版本号，则说明安装成功
